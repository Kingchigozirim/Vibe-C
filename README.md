Features

Responsive Design — Works seamlessly on desktop and mobile devices.

Gradient and Accent Styling — Modern color palette with gradient text and smooth hover effects.

Smooth Scroll — Button links scroll smoothly to page sections.

Clean Layout — Flex-based layout with clear sections: header, main, footer.

Scalable Typography — Uses clamp() for responsive headings.

Vibe Coding Style — Emphasis on intention-driven design and AI-assisted development workflow.

Technologies Used

HTML5 — Semantic structure for accessibility and SEO.

CSS3 — Variables, Flexbox, gradients, and responsive design.

JavaScript — Smooth scroll effect for anchor links.

Google Fonts — Inter font for modern, readable typography.
